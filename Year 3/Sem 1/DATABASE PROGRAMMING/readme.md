## Reflection

Database Programming extended the database knowledge I gained in Year 2 by focusing on database implementation and application integration. Through assignments and the Event Booking System project, I learned how databases support modern applications and how different database technologies can be used to manage data efficiently.

The project provided practical experience in designing data models, implementing CRUD operations, and working with JSON collections. It also introduced me to NoSQL concepts and broadened my understanding of modern database technologies.

This course strengthened my technical skills in database development and reinforced my interest in pursuing a career related to data engineering and database systems.

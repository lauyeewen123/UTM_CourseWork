## 📚 Key Concepts Learned

### Distributed Computing

Apache Spark distributes workloads across multiple computing resources, allowing large datasets to be processed efficiently.

### In-Memory Processing

Spark improves performance by processing data in memory rather than relying heavily on disk operations.

### Scalable Data Engineering

Spark provides a scalable framework for handling large-scale data transformation and analytics workloads.

### Data Modelling

The implementation of a star schema demonstrated how structured analytical models support reporting and decision-making processes.

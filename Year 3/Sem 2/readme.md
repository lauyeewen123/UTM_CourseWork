## 🌟 Featured Projects

### TVETMARA Student Attendance Management System
**Course:** Application Development

- Developed a mobile attendance management application using Flutter and Firebase.
- Implemented attendance recording, discipline reporting, timetable management, and user management features.
- Applied mobile application development, cloud database integration, and software testing practices.

### PPG Recoverable Assets & Inventory Risk Management (RA & IRM)
**Course:** Special Topic in Data Engineering

- Designed a Galaxy Schema data warehouse model to support business analytics requirements.
- Developed analytical data structures using Azure Synapse Analytics and cloud-based data engineering tools.
- Built interactive Power BI dashboards to analyze inventory health, stock shortages, recoverable assets, dead stock, and customer impact.
- Applied Azure Data Factory, Azure Data Lake Storage, Azure Databricks, Azure Synapse Analytics, and Power BI.

### High Performance Data Processing Optimization Project
**Course:** High Performance Data Processing

- Processed large datasets using Pandas, Polars, and DuckDB.
- Evaluated performance optimization techniques for scalable data processing.
- Compared processing efficiency across different analytical approaches.

### AI-Assisted Data Engineering Pipeline
**Course:** Special Topic in Data Engineering

- Developed an AI-assisted data quality workflow using Apache Airflow and Airia AI.
- Implemented automated data validation, monitoring, and workflow orchestration.
- Gained practical experience in modern data engineering practices.

### Business Intelligence Dashboard Projects
**Course:** Business Intelligence

- Developed interactive Power BI dashboards for data analysis and decision-making.
- Applied data modelling, KPI design, and visualization best practices.
- Generated business insights through analytical reporting.

## Reflection

This course provided valuable exposure to large-scale data processing and performance optimization techniques. Through the JobStreet Malaysia analytics project, I learned how to collect, clean, process, and optimize datasets using modern data engineering tools.

One of the most important lessons was understanding how different technologies can affect processing performance. Comparing Pandas, Polars, and DuckDB helped me appreciate the importance of selecting the right tool for specific workloads.

This course strengthened my technical skills in data engineering and provided practical experience that closely resembles real-world industry workflows.

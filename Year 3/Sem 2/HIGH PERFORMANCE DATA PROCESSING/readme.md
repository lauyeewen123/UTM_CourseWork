## 🚀 Featured Project: JobStreet Malaysia Analytics Pipeline

### Project Objective

The objective of this project was to collect, process, and analyse job vacancy data from JobStreet Malaysia while applying high-performance data processing techniques to improve efficiency and scalability.

### Key Contributions

- Designed the system architecture and data processing workflow.
- Developed and optimized data processing pipelines.
- Evaluated performance using different processing frameworks.
- Compared processing efficiency between Pandas, Polars, and DuckDB.
- Generated insights from large-scale job market data.

## 💡 Reflection

This course provided valuable exposure to the challenges of processing large datasets efficiently. Through assignments and the JobStreet Malaysia Analytics Pipeline project, I learned how modern data engineering workflows are designed, optimized, and evaluated.

One of the most interesting aspects of the course was comparing different data processing technologies such as Pandas, Polars, and DuckDB. This helped me understand that selecting the right tool can significantly impact performance, scalability, and resource utilization.

The project also strengthened my understanding of data collection, cleaning, transformation, and optimization processes. These experiences provided practical insights into real-world data engineering workflows and further reinforced my interest in pursuing a career in Data Engineering.

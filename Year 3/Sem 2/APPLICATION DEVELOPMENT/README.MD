## Reflection

Application Development provided practical experience in building modern software applications. Through laboratory exercises and project development, I learned how to design user-friendly interfaces, manage application logic, and integrate databases into software systems.

This course improved my understanding of the complete software development lifecycle, from planning and implementation to testing and deployment. It also strengthened my teamwork and problem-solving skills through collaborative project work.

Overall, the course enhanced my confidence in developing real-world applications and applying software engineering principles in practical environments.

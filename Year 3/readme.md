## 🌟 Featured Projects

Some of the major projects completed during Year 3 include:

### PPG Recoverable Assets & Inventory Risk Management (RA & IRM)
- Designed a Galaxy Schema data warehouse model.
- Developed analytical pipelines using Azure technologies.
- Built Power BI dashboards to support inventory and business decision-making.
- Applied Azure Synapse Analytics, Azure Data Factory, and Databricks.

### TVETMARA Student Attendance Management System
- Developed a Flutter and Firebase-based attendance management application.
- Implemented attendance tracking, discipline reporting, and timetable management features.
- Applied mobile application development and cloud database integration techniques.

### High Performance Data Processing Project
- Performed large-scale data processing and optimization using Pandas, Polars, and DuckDB.
- Evaluated performance improvements through different processing techniques and architectures.

### AI-Assisted Data Engineering Pipeline
- Developed an Airflow-based data quality pipeline integrated with AI-assisted validation techniques.
- Applied workflow orchestration and automated data quality monitoring.

### Business Intelligence & Data Analytics Projects
- Designed interactive dashboards using Power BI.
- Applied data visualization techniques to generate business insights and support decision-making.

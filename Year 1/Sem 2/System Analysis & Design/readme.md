## Reflection

System Analysis and Design taught me that successful software projects require more than just programming skills. Through assignments and project work, I learned how to gather requirements, analyse business processes, and create system models that support software development.

The project helped me understand the importance of planning, documentation, and communication during the software development lifecycle. This course strengthened my ability to bridge business requirements and technical solutions, which is an essential skill in software engineering and data-related projects.

## Key Learning Outcomes

### 🔹 Computer Organization & Architecture
- Learned the internal organization of computer systems, including processors, memory hierarchy, instruction execution, and computer performance.
- Developed an understanding of how hardware components interact to execute programs efficiently.

### 🔹 Probability & Statistical Data Analysis
- Applied probability concepts and statistical techniques to analyse and interpret data.
- Learned descriptive statistics, probability distributions, hypothesis testing, and data-driven decision making.

### 🔹 Programming Technique II
- Enhanced programming skills through object-oriented programming concepts and advanced problem-solving techniques.
- Developed larger and more structured programs using modular design principles.

### 🔹 System Analysis & Design
- Learned the System Development Life Cycle (SDLC), requirements gathering, process modelling, and system design methodologies.
- Applied UML diagrams and analysis techniques to design information systems that meet business requirements.

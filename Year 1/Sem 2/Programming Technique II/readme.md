## Reflection

Programming Technique II expanded my programming knowledge beyond basic procedural programming by introducing Object-Oriented Programming concepts. Through exercises and projects, I learned how to develop programs using classes, objects, inheritance, and encapsulation.

The group project provided valuable experience in software design and teamwork. It taught me how to organize code more effectively and develop larger applications using reusable components. This course laid the groundwork for future subjects such as Software Engineering, System Development Technology, and Application Development.

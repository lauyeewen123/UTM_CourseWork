## 🗂 Academic Journey

### 📖 Year 1 – Computing Fundamentals

Built a strong foundation in programming, mathematics, computer systems, and information technology through courses such as Programming Technique, Digital Logic, Discrete Structure, and Computer Organization & Architecture.

### 💻 Year 2 – Software Development & Data Analytics

Expanded my knowledge in software engineering, databases, networking, operating systems, data mining, and analytics. During this stage, I completed several system development and database design projects while strengthening my programming skills.

### 📊 Year 3 – Data Engineering Specialization

Focused on advanced topics including data engineering, artificial intelligence, database programming, application development, cloud technologies, and high-performance data processing. These courses provided practical exposure to modern industry tools and technologies.

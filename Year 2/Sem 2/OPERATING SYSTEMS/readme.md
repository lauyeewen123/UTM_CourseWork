## Reflection

Operating Systems helped me understand how software interacts with hardware resources and how operating systems manage processes, memory, and system performance.

The scheduling project provided valuable insights into CPU scheduling algorithms such as Shortest Job First (SJF) and Priority Scheduling. Through practical exercises, I learned how operating systems optimize resource utilization and system efficiency.

This course strengthened my understanding of system-level computing concepts that are essential in software engineering, cloud computing, and data engineering environments.

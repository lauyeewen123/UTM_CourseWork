## Reflection

Data Analytics Programming was one of the courses that strengthened my interest in data engineering and analytics. Through assignments and projects, I learned how to collect, clean, transform, and visualize data using Python.

Working with real datasets helped me understand the importance of data quality and preparation before analysis. I also gained practical experience using Pandas and visualization libraries to generate insights from data.

This course provided a strong foundation for future subjects such as Data Mining, Business Intelligence, and Data Engineering.

Welcome to my Year 2 Semester 2 coursework portfolio. This folder contains assignments, projects, laboratory exercises, reports, and learning materials completed during the second semester of my Bachelor of Computer Science (Data Engineering) programme at Universiti Teknologi Malaysia (UTM).

## Key Learning Outcomes

### 🔹 Data Analytics Programming
- Applied programming techniques for data analysis and manipulation.
- Utilized data processing libraries and tools to clean, transform, and analyze datasets.
- Developed practical skills in extracting meaningful insights from data.

### 🔹 Data Mining
- Learned techniques for discovering patterns, trends, and relationships within large datasets.
- Applied classification, clustering, association rule mining, and predictive analytics methods.
- Gained experience in data preprocessing, model evaluation, and result interpretation.

### 🔹 Object Oriented Programming
- Applied object-oriented programming principles such as encapsulation, inheritance, polymorphism, and abstraction.
- Developed larger and more maintainable software applications using structured design approaches.
- Enhanced software development and debugging skills through practical projects.

### 🔹 Operating Systems
- Studied operating system concepts including process management, memory management, file systems, scheduling, and concurrency.
- Learned how operating systems manage hardware resources and support application execution.
- Developed a deeper understanding of system-level computing concepts.

## 💡 Reflection

Year 2 Semester 2 strengthened my programming, analytical, and problem-solving skills through hands-on coursework and projects. The combination of data analytics, data mining, object-oriented programming, and operating systems provided a balanced understanding of both software development and data-driven technologies.

The practical assignments and projects helped me improve my ability to analyze complex problems, develop efficient solutions, and work with real-world datasets. These experiences further reinforced my interest in Data Engineering and prepared me for more advanced topics such as cloud computing, big data processing, business intelligence, and machine learning in Year 3.

## Reflection

This course deepened my understanding of object-oriented software development. Using Java, I learned how to design systems using classes, objects, inheritance, and polymorphism.

The Event Management System mini project provided practical experience in applying OOP principles to develop a complete application. It also improved my ability to structure programs effectively and manage larger codebases.

The concepts learned in this course continue to be useful in software engineering and application development projects.

## Reflection

This course provided practical exposure to computer networking concepts and communication protocols. Through Packet Tracer simulations and laboratory exercises, I learned how devices communicate within a network and how network services operate.

The hands-on activities improved my understanding of network design, routing, switching, and protocol behaviour. This knowledge helps me better understand the infrastructure that supports modern information systems and cloud environments.

## Reflection

Software Engineering taught me how professional software projects are planned and managed. Through the Technicrab project, I learned how to gather requirements, define system scope, and document software specifications.

Developing the Software Requirements Specification (SRS) improved my ability to communicate technical requirements clearly and systematically. The course emphasized teamwork, documentation, and project organization, which are essential skills for software development projects.

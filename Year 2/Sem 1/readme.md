## Key Learning Outcomes

### 🔹 Data Structure and Algorithm
- Learned fundamental data structures such as arrays, linked lists, stacks, queues, trees, and graphs.
- Applied algorithm design and analysis techniques to solve computational problems efficiently.
- Improved problem-solving and logical thinking skills through coding exercises and projects.

### 🔹 Database (WBL)
- Designed and implemented relational databases using SQL.
- Applied database normalization, entity-relationship modelling, and query optimization techniques.
- Developed database solutions to support business requirements and data management processes.

### 🔹 Network Communications
- Learned the principles of computer networking, communication protocols, and network architecture.
- Explored topics such as TCP/IP, routing, switching, network security, and internet technologies.
- Gained practical knowledge of network configuration and troubleshooting.

### 🔹 Software Engineering (WBL)
- Applied software development methodologies and engineering principles throughout the software development life cycle (SDLC).
- Learned requirements analysis, software design, testing, maintenance, and project management practices.
- Developed teamwork and collaboration skills through group-based software projects.

### 🔹 System Development Technology
- Gained hands-on experience in developing information systems and web-based applications.
- Applied system analysis, design, implementation, testing, and deployment techniques.
- Strengthened technical skills through real-world development projects and documentation activities.

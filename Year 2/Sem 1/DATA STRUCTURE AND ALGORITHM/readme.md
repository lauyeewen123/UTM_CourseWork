# 🌳 Data Structure and Algorithm

## Course Overview

This course introduced fundamental data structures and algorithms used to solve computational problems efficiently. Topics included arrays, linked lists, stacks, queues, recursion, searching, sorting, and algorithm analysis.

## Reflection

This course significantly strengthened my problem-solving abilities by introducing efficient ways to organize and process data. Through assignments and laboratory exercises, I learned how different data structures can impact the performance of a program.

The mini project provided practical experience in applying algorithms to solve real-world problems. I also developed a better understanding of computational efficiency and algorithm optimization. These concepts continue to be valuable in software development, database systems, and data engineering applications.

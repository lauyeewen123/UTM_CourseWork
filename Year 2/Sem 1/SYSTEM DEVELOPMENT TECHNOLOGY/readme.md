## Reflection

System Development Technology was one of the most practical courses I have completed. Through the KADA Management System project, I experienced the complete software development process, from requirement analysis and design to implementation and testing.

Developing a functional web application improved my programming, database management, and problem-solving skills. I also gained valuable experience working in a development team and managing project deliverables. This course significantly strengthened my confidence in developing real-world software systems.
